\doxysection{exemplo\+\_\+2.\+c}
\hypertarget{exemplo_2_8c-example}{}\label{exemplo_2_8c-example}Sistema de Datalogger CSV com busca por coluna para microcontroladores AVR.

Este exemplo demonstra como criar um arquivo CSV, gravar múltiplos registros simulando sensores e realizar buscas inteligentes por valores em colunas específicas, tudo otimizado para o compilador XC8 e baixa utilização de memória RAM.
