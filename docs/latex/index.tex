Bem-\/vindo à documentação oficial da biblioteca {\bfseries{AVR SD Card}}. Este driver foi desenvolvido para permitir que microcontroladores da linha AVR (como o ATmega328P) manipulem arquivos em cartões SD formatados em {\bfseries{FAT32}}.\hypertarget{index_autotoc_md6}{}\doxysection{\texorpdfstring{Principais Recursos}{Principais Recursos}}\label{index_autotoc_md6}

\begin{DoxyItemize}
\item {\bfseries{Baixo Consumo de RAM:\+}} Utiliza buffers otimizados para operar em apenas 2KB de SRAM.
\item {\bfseries{Streaming de Dados:\+}} Funções de leitura e escrita via callback para processamento em tempo real.
\item {\bfseries{Gerenciamento FAT32:\+}} Suporte a criação, exclusão, append e expansão automática de diretórios.
\item {\bfseries{Documentação Completa:\+}} Grafos de chamadas gerados automaticamente para facilitar o entendimento do fluxo SPI/\+\+FAT.
\end{DoxyItemize}\hypertarget{index_autotoc_md7}{}\doxysection{\texorpdfstring{Pinagem Sugerida (ATmega328P)}{Pinagem Sugerida (ATmega328P)}}\label{index_autotoc_md7}
