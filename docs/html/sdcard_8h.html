◆ sd_fat_mount()

SD_Status sd_fat_mount ( SD_Context * ctx,
FAT32_Info * fs )

Localiza e monta o sistema de arquivos FAT32 no cartão SD.

Realiza a leitura do MBR para localizar a primeira partição primária. Em seguida, extrai os parâmetros do BPB (BIOS Parameter Block): setores por cluster, setores reservados, localização das FATs e do diretório raiz. Inicializa também os caches internos de diretório e FAT para operações O(1).

Parâmetros
ctxPonteiro para o contexto da biblioteca SD inicializado.
fsPonteiro para a estrutura FAT32_Info que será preenchida.
Retorna
SD_Status SD_SUCCESS, SD_ERR_MBR (assinatura inválida) ou SD_ERR_BOOT.

◆ sd_file_read()

SD_Status sd_file_read ( SD_Context * ctx,
FAT32_Info * fs,
const SD_FileName * file,
uint8_t * buffer,
uint32_t buffer_size )

Lê o conteúdo de um arquivo FAT32 para um buffer externo.

Esta função realiza a leitura completa de um arquivo, percorrendo toda a cadeia de clusters (cluster chain) na FAT32.

Os dados são copiados sequencialmente para o buffer fornecido pelo usuário.

Parâmetros
ctxPonteiro para o contexto da biblioteca SD
fsPonteiro para a estrutura do sistema FAT32 montado
filePonteiro para estrutura contendo nome do arquivo (formato 8.3)
bufferBuffer externo onde os dados sero armazenados
buffer_sizeTamanho mximo do buffer em bytes
Retorna
  • SD_SUCCESS: leitura realizada com sucesso
  • SD_ERR_NOT_FOUND: arquivo não encontrado
  • SD_ERR_READ: erro durante leitura de bloco ou cluster
  • SD_ERR_DISK_FULL: buffer insuficiente para armazenar o arquivo

◆ sd_quick_init()

SD_Status sd_quick_init ( SD_Context * ctx)

Inicialização rápida para AVR (configura pinos + HAL interno).

Configura automaticamente:

  • Pinos PB2=CS, PB3=MOSI, PB5=SCK, PB4=MISO
  • SPI Mestre, modo 0
  • HAL interno (transfer, cs_low, cs_high)
Observação
O delay_ms deve ser injetado antes ou via sd_begin()

◆ sd_set_filename()

SD_Status sd_set_filename ( SD_FileName * file_out,
const char * name_in )

Converte uma string convencional para o formato FAT 8.3.

Limpa a estrutura de destino com espaços e realiza a conversão automática para caracteres maiúsculos. O nome é limitado a 8 caracteres e a extensão a 3. Caracteres especiais e pontos extras são ignorados conforme o padrão FAT.

Parâmetros
file_outPonteiro para a estrutura SD_FileName que será preenchida.
name_inString de entrada (ex: "LOG_2026.txt" vira "LOG_2026" e "TXT").
Retorna
SD_Status SD_SUCCESS ou SD_ERR_READY se os ponteiros forem nulos.
