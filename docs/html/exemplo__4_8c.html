◆ delay_ms_variavel()

void delay_ms_variavel ( uint16_t ms)

Executa um delay variável em milissegundos.

Como a macro _delay_ms() do AVR exige um valor constante em tempo de compilação, usamos esta função que repete o delay de 1ms n vezes, permitindo usar o valor lido do cartão SD.

Parâmetros
msQuantidade de milissegundos para aguardar.
